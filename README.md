# Cyfuture AI Chatbot

This repository contains the source code for the Cyfuture AI Chatbot.  
The chatbot is designed to handle customer queries, automate responses, and integrate with Cyfuture AI services.

## Features
- Natural language understanding
- Multi-channel support (web, WhatsApp, API)
- Custom conversation flows
- Integration with Cyfuture AI backend and GPU cloud
- Easy deployment on web or internal systems

## Tech Stack
- Node.js / Python (choose based on your implementation)
- Express / FastAPI
- Webhook or REST API integration
- Chat UI (optional)
